# obsidian

Шаблоны и заметки для [Obsidian](https://obsidian.md/).

## templates/

| Файл | Назначение |
|---|---|
| [`Торговый дневник.md`](templates/Торговый%20дневник.md) | Ежедневный торговый/инвестиционный дневник: совершённые сделки, упущенные сделки (через 24 ч) и вечерняя рефлексия. |

### Требования

Шаблон использует плагин **Templater** (синтаксис `<% ... %>`), а не встроенные Core-шаблоны.

1. Установите плагин Templater: *Настройки → Сторонние плагины → Templater*.
2. В настройках Templater укажите папку шаблонов (например, `obsidian/templates`).
3. Создавайте заметку через *Templater → Create new note from template* и выберите «Торговый дневник».

При создании заметка автоматически:

- перемещается в папку `инвестиции/дневник/`;
- получает имя по дате создания (`DD.MM.YYYY`);
- заполняет frontmatter (`date`, `type`, `tags`).

> [!note] Папка `инвестиции/дневник/` должна существовать в хранилище (vault) — Templater не создаёт промежуточные папки.
