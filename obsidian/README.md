# obsidian

Шаблоны и заметки для [Obsidian](https://obsidian.md/).

## templates/

| Файл | Назначение |
|---|---|
| [`Торговый дневник.md`](templates/Торговый%20дневник.md) | Ежедневный торговый/инвестиционный дневник: совершённые сделки, упущенные сделки (через 24 ч) и вечерняя рефлексия. Папка: `инвестиции/дневник/`. |
| [`Отчёт за день.md`](templates/Отчёт%20за%20день.md) | Ежедневный отчёт сисадмина: дата, клиенты и выполненные работы (нескольких клиентов и работ). Папка: `отчёты за день/`. |

### Требования

Шаблон использует плагин **Templater** (синтаксис `<% ... %>`), а не встроенные Core-шаблоны.

1. Установите плагин Templater: *Настройки → Сторонние плагины → Templater*.
2. В настройках Templater укажите папку шаблонов (например, `obsidian/templates`).
3. Создавайте заметку через *Templater → Create new note from template* и выберите «Торговый дневник».

При создании заметка автоматически:

- создаёт целевую папку (`инвестиции/дневник/` или `отчёты за день/`), если её ещё нет;
- перемещается в эту папку;
- получает имя по дате создания (`DD.MM.YYYY`); при конфликте имени добавляется время (`HH-mm`);
- заполняет frontmatter (`date`, `type`, `tags`).
